# 前端的思考- 终章
随着时间的发展和对业务的理解和程序的理解，越发觉得前端的天花板其实很低，本身是不具备任何技术门槛。即便是最为天花板的编辑器，其本身也是存在了不少的问题。

还是那句话: 先学会用，再去做。不要把自己的时间浪费在一些不必要的事情上。
